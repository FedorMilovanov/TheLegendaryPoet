# Маяковский и Брики — визуальный пакет

Рабочая ветка: `work/mayakovsky-visual-essays`.

## Реализовано

- Двухчастная биография Владимира Маяковского.
- Отдельный визуально-документальный материал о Лиле и Осипе Брик.
- Типизированный блок архивного изображения.
- Минималистичные подписи с маркировкой: архив / документ / цифровая реставрация / художественная реконструкция.
- 3D Tilt, полноэкранный просмотр, Escape и клик по фону для закрытия.
- Серийная навигация между частями биографии.
- Проверка обязательных полей изображения в `validate-essays`.

## Правила публикации

1. Архивный оригинал не заменяется реставрацией.
2. Генеративно восстановленная версия маркируется как цифровая реставрация.
3. Реконструкция никогда не подписывается как архивная фотография.
4. Для каждого архивного кадра обязательны `alt`, короткая подпись, источник и тип изображения.
5. Веб-версии готовятся в WebP; полноразмерные исходники хранятся отдельно от публичного ассета.
6. Никакой колоризации без отдельного редакционного решения.
