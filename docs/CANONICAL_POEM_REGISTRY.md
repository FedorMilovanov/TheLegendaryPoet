# CANONICAL_POEM_REGISTRY.md — реестр сверки текстов стихов

Последний раз обновлено: 2026-07-16. Ведёт: Фёдор Милованов / Arena.

## Назначение

Единое место, где **каждый канонический текст стиха в библиотеке** привязан к
верифицированным первоисточникам. Цель проекта — сверять каждое стихотворение
минимум по двум независимым источникам, хотя бы один A+ (см. иерархию в
`PROJECT_CHARTER.md`, `POET_AUTHORING_GUIDE.md` §11).

**Правила ведения:**
- При добавлении нового стиха — сразу заноси его сюда со ссылками.
- При смене редакции текста — обнови ссылки и отметь дату.
- Мёртвая ссылка = регресс: проверь и замени или удали.
- Уровни: **A+** (автограф / академ. ПСС / ФЭБ / РВБ / Библ. Поэта со ссылкой на
  автограф / первая публ.), **A** (академ. Викитека, culture.ru, сайт-музей,
  монография ЖЗЛ), **B** (журналистика со ссылкой на архив).

## Орфографическая оговорка

В ряде стихов допускается вариант «е/ё» (напр. «Поймёт/Поймет», «далёко/далеко»,
«тёмной/темной»). Проект не требует жёсткой унификации «ё», но хранит текст в
том виде, в каком он обычно приводится в академических изданиях; расхождение
«е/ё» НЕ считается ошибкой. Это не касается собственно слов и пунктуации —
там сверка строгая.
