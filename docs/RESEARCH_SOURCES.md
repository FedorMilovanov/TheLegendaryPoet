# Исследовательские источники

Первоисточники и авторитетные ресурсы, по которым сверялись тексты стихотворений и
факты биографий. Правило проекта: **каждый канонический текст — сверять минимум по
двум независимым источникам**, опираться на первоисточники (академические издания,
Викитека, РВБ, ФЭБ), а не на пересказы. Ниже — рабочий список (пополняется).

## Тексты стихотворений (сверка)

- Викитека (ru.wikisource.org) — академические редакции
- РВБ — Русская виртуальная библиотека (rvb.ru)

- rustih.ru, slova.org.ru — сверочные тексты
